class DicePP.data_manager.data_chunk.DataChunkBase[源代码]¶

DataChunk是一次读取/更新文件的最小单位, 每个DataChunk子类都对应一个同名的持久化json文件 为了方便阅读和管理, 一个DataChunk应当包括某一类功能所需要的全部数据, 也不应包含太多或太少内容 不能拥有非基础类型, 自定义类型必须继承自DataManager.JsonObject! 否则无法序列化 例子: 保存20000条某类数据, 每条数据200字节, 大概就是4MB
